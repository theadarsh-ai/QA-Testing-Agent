DesignGuard is a full-stack autonomous QA testing agent that performs complete visual and backend quality assurance on any web application with zero setup and zero manual test scripts. A developer simply pastes their application URL — whether it's running on localhost or deployed to production — and DesignGuard's LangGraph-powered agent takes over completely. Using Playwright as its hands, the agent autonomously opens the application in a headless browser, navigates through every page by clicking links, buttons, menus, and tabs by itself, and takes a screenshot after every action. Every screenshot is sent to Gemini 2.0 Flash which reads the screen purely visually — like a senior QA engineer — and detects visual bugs including broken layouts, text overflow, inconsistent fonts and colors, misaligned elements, missing images, and responsive design failures at different screen sizes. Simultaneously, Playwright intercepts every network request the application makes, capturing all API calls, status codes, response times, and error responses — giving DesignGuard full backend visibility without any server-side integration. The agent also compares the built application against an uploaded Figma design mockup to catch pixel-level design deviations, supports voice commands so developers can say "check the login form" and the agent navigates directly to it, generates a before/after visual diff to catch regressions between deployments, and checks design consistency across all pages simultaneously. Firestore stores every scan permanently, so the next time the same URL is scanned, DesignGuard automatically detects new bugs that appeared since the last scan. The entire pipeline — Observe → Navigate → Detect → Fix → Verify → Report — runs as a LangGraph StateGraph agent hosted on Google Cloud Run, uses the Google GenAI SDK for all Gemini calls, and delivers a final professional PDF QA report with every bug screenshot, severity rating, exact CSS/HTML code fix, API fix suggestion, performance metrics, and an overall quality score — all from a single URL input, in under 60 seconds.
